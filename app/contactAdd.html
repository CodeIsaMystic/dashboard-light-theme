Ajouter des contacts et témoignages

Dans cette partie, vous pouvez ajouter un contact à votre liste, enregistrer un nouveau témoignage que vous pourrez activer par la suite en cochant la case active et choisissant aussi sa position sur votre site web. Vous avez trois "cards" témoignages, vous pouvez alors choisir la "card" 1, 2, 3 et dès lors que vous enregistrer les changements seront publiés sur votre site web.

Notez que tout vos changements seront stockés dans votre base de données dès lors que vous les aurez enregistrés. Vous pouvez également les consultés via les listes sur votre dashboard, à la page correspondante.

Ajouter des contacts

Vous avez ici la possibilité d'enregistrer vos contacts dans votre base de données. Les contacts acquis via le site web principal seront également stockés dans cette liste. Une seule liste pour tout vous contacts, ce sera le moyen de vous créer une liste d'email et de pouvoir ainsi contacter vos prospects.

Ajouter des témoignages

Dans cette partie, vous pouvez ajouter un contact, un témoignage à leur liste respective puis enregistrer dans votre base de données.
