# Dashboard Light Theme


## Getting Started

This app is a basic backend dashboard for simple projects.The [dashboard-light-theme](https://github.com/CodeIsaMystic/dashboard-light-theme) will pretend having various uses cases with some basics and standards applications, you can imagine so, handling texts, SEO, Titles, Cards, Images, Users, Contacts, Lists, Articles, Posts and so on...
You can of course implementing more applications on it like charts with Chart.js.
