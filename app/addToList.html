Ajout de contact & témoignage

Dans cette partie, vous pouvez ajouter un contact à votre liste et enregistrer un nouveau témoignage Notez que tout vos changements seront stockés dans votre base de données dès lors que vous les aurez enregistrés. Vous pouvez également les consultés via les listes sur votre dashboard, à la page correspondante.

Ajouter des contacts

Vous avez ici la possibilité d'enregistrer vos contacts dans votre base de données. Les contacts acquis via le site web principal seront également stockés dans cette liste. Une seule liste pour tout vous contacts, ce sera le moyen de vous créer une liste d'email et de pouvoir ainsi contacter vos prospects.

Ajouter des témoignages

Avec ce formulaire, vous ajoutez un nouveau témoignage que vous pourrez publier en cochant la case "Publiez" et choisir aussi sa position sur votre site web. Vous avez trois "cards" témoignages, vous pouvez alors choisir la "card" 1, 2, 3 et dès lors que vous enregistrer les changements seront publiés sur votre site web.
